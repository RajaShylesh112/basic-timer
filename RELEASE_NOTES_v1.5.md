# BasicTimer v1.5 - New Features Release

## 🎉 New Features

### 1. **Session History Tracking** 📊
- **Automatic tracking** of all timer sessions longer than 30 seconds
- **Detailed history** showing start time, end time, duration, and activity
- **Daily and weekly summaries** of time spent
- **Export to CSV** for external analysis
- **Access via**: Right-click menu → "Session History"

### 2. **Quick Timer Presets** ⚡
- **Instant timer setup** with popular durations
- **Work & Focus presets**: 5min, 10min, 15min, 25min (Pomodoro), 30min, 45min
- **Break presets**: 2min, 5min (short break), 15min (long break)
- **Extended sessions**: 1hr, 1.5hr, 2hr
- **Pomodoro-friendly** with highlighted 25-minute preset
- **Access via**: Right-click menu → "Quick Presets"

### 3. **Enhanced Time Display**
- **Hour display toggle**: Switch between MM:SS and HH:MM:SS formats
- **Access via**: Right-click menu → "Show Hours"

### 4. **Inline Time Editing**
- **Double-click the timer** to edit time directly
- **No dialog boxes** - edit in place
- **Keyboard shortcuts**: Enter to save, Escape to cancel
- **Smart parsing**: Supports both MM:SS and HH:MM:SS formats

### 5. **Improved Defaults**
- **Count up by default** (more intuitive for most users)
- **Starts at 00:00** instead of 30:00
- **Better dialog behavior** with proper OK/Cancel handling

## 🚀 Quick Start Guide

### Setting Quick Timers:
1. Right-click the timer
2. Choose "Quick Presets"
3. Click any preset button (e.g., "25 Minutes (Pomodoro)")
4. Timer is instantly set and ready to start

### Viewing Session History:
1. Right-click the timer
2. Choose "Session History"
3. View your time tracking data, export to CSV, or clear history

### Inline Editing:
1. Double-click directly on the timer display
2. Type new time (e.g., "25:00" or "1:30:45")
3. Press Enter to set, or Escape to cancel

## 📋 Session Data Export

Session history can be exported as CSV with these columns:
- Start Time
- End Time  
- Duration
- Type (Count Up/Countdown)
- Initial Time
- Activity (based on timer title)

## 🔧 Technical Notes

- Sessions shorter than 30 seconds are not recorded
- Session tracking starts when timer begins and ends when stopped
- Data is stored in memory during the session (not persisted between app restarts)
- All new features maintain backward compatibility
